# Example Custom WordPress Theme

This repository contains an example custom WordPress theme created as a
code sample for recruitment purposes.

## About the project

The theme was built from scratch without page builders, based on real-world
commercial and personal projects. It focuses on clean architecture,
maintainable code, and flexibility for content editors.

## Key features

- Custom WordPress theme built from scratch
- Advanced Custom Fields (ACF) including Flexible Content and modular sections
- Flexible and modular template structure
- SCSS with BEM methodology
- Responsive layout
- Performance-oriented approach

## Tech stack

- WordPress
- PHP
- Advanced Custom Fields (ACF)
- JavaScript
- SCSS (BEM)
- HTML5

## Project structure

The repository contains only theme-related files.
WordPress core files and client content are intentionally excluded.

Key logic is located in the /inc directory and modular templates
inside /template-parts.

## Relation to live projects

This theme structure is based on real projects available online:

- https://trenerpersonalnykrosno.pl  
- https://kiwidjs.pl  

## Notes

This repository does not represent a full WordPress installation.
It is intended to showcase code quality, structure, and development approach.
